  • Originally built as part of Temple of Pele, Skyrmion has been converted into a full 2d game engine based on SFML.
    It has a focus on tilemaps and grids, but also supports dynamic lighting, remappable controls, and more.
    Details and some documentation in the readme.

  • This is a WIP Solitaire card game engine with themes and joystick controls.
    It uses a custom programming language based on Lisp to allow adding new card games on runtime. Progress has been made towards allowing multiplayer games and auto-solving tips.

  • Temple of Pele

    Built as a group project in college, this is a simple 2d puzzle dungeon game.
    It has also become a test bed for new features in the Skyrmion engine.

  • One of my many solo game jams, a short top down coop game.
    This is where I made a dynamic lighting engine for Skyrmion.

  • The first game jam I did with a team, the first side-scrolling game using Skyrmion.
    Required creating a custom physics engine and improving large world mapping.
